2.01背包问题

题目

有 N 件物品和一个容量是 V 的背包。每件物品只能使用一次。

第 i 件物品的体积是 vi，价值是 wi。

求解将哪些物品装入背包，可使这些物品的总体积不超过背包容量，且总价值最大。
输出最大价值。

输入格式

第一行两个整数，N，V，用空格隔开，分别表示物品数量和背包容积。

接下来有 N 行，每行两个整数vi，wi，用空格隔开，分别表示第 i件物品的体积和价值。

输出格式

输出一个整数，表示最大价值。

数据范围

0<N,V≤1000
0<vi,wi≤1000

解题思路

01背包考查得是状态转移，动态规划问题。

状态转移，即当前状态依赖于前一个状态。

那么在本题中，当前装入入物品的背包状态（价值最优解）根据上一个物品的背包状态而决定。

01背包是对第i个物品做出决策，01代表不选和选的决定。

二维版本

  • 状态方程f[i][j]定义：前i个物品(包含第i个物品)，背包j体积下的价值最优解。
  • 当背包装不进第i个物品：j < v[i]，那么最优解就是上一个物品i-1在背包j体积下的最优解。
    • f[i][j] = f[i-1][j]
  • 当背包能够装进第i个物品，两个决定去最大值，max(不选，选)。
    • 不选：f[i][j] = f[i-1][j]
      • 不选该物品，为什么会有可能得到最优解？很简单，因为要装入第i个物品，那么就要空出背包体积v[i]，空出背包导出缺失的物品价值，不一定小于第i个物品的价值。
    • 选：f[i][j] = f[i-1][j-v[i]] + w[i]
      • 根据上一个物品的背包状态，f[i-1][j]，那么要装入第i个物品，即要找到f[i-1][?]在j-v[i]时的状态（f[i-1][j-v[i]]），加上当前第i个物品的价值，即是当前装入第i个物品时背包的最优解。

二维的状态转移方程：f[i][j] = max(f[i-1][j], f[i-1][j-v[i]] + w[i])

从f[0][0]=0开始决策，有N个物品，就决策N次，选与不选，选其中最大值，那么f[i][j]的状态在不停地更新迭代，当迭代到最大值时，即f[N][V]为最优解。

一维

题目要求得到容量j的背包价值最优解，那么和物品无关，只与背包容量j相关，用f[j]也能得到最优解。

为什么和物品无关？

可以这么理解，背包只负责价值最优解，那么有N件物品，不一定全部用到，毕竟背包容量一定，所以装入N件物品的最优解和装入N-1（数量不一致）件物品的最优解可能是一样，装入物品的种类和数量一样，只是可选择的物品种类数不一样而已。所以容量一定的背包价值最优解，和物品无关，只与背包容量相关。

换言之，将f[i][j]等价到f[j]。

如何做到呢？

最优解只由j决定。定义如下：

状态f[j]定义：N件物品，背包容量j下的最优解。

在二维情况下，状态f[i][j]由上一轮（循环到第i个物品）的f[i-1][j]更新而来。

而在一维情况下，状态f[j]其实也是由上一轮的f[j]更新而来，即f[i-1][j]，如果决策能装的下，即f[j]由f[j-v[i]]更新。

但是有个问题，一维状态下背包容量m是升序遍历的话，那么如果在当前这轮中，决策体积为3的第i个物品时， f[7]由f[4]更新而来，那么在上一次更新f[4]时，其实在当前轮次更新，并非i-1轮次，那么就有可能出现一种情况，更新f[4]已经将体积为3的第i个物品放入背包，而在当前更新f[7]时，又将第i个物品放入背包，那么与题目要求不符，每个物品只能用一次。

但是，只有由i-1轮更新的f[4]才是应该去更新当前第i轮的f[7]，所以如何避免在第i轮更新f[4]呢？

很简单，只需要将背包容量m降序遍历即可，当前第i轮的f[7]需要f[4]时，而因为降序遍历，此时当前第i轮f[4]还没更新，即第i轮的f[4]的最优解仍然是第i-1轮的f[4]。

一维的状态转移方程为：f[j] = max(f[j], f[j - v[i]] + w[i])
