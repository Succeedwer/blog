导入已有的ubuntu

导入已有的导出tar包

  • Ubuntu: 需要导入的linux名称
  • D:/wsl/Ubuntu: 导入后存储ext4.vhdx的路径
  • D:/wsl/Ubuntu.tar: 需要导入的tar或vhdx文件
  • --version 2 : 使用的wsl版本

导入已有的ext4.vhdx

  1. 将已有的linux迁移，如果wsl没有安装任何linux，可以先安装一个新的linux。

  2. 将已有的ext4.vhdx替换D:\wsl\Ubuntu\ext4.vhdx。

  3. 进入系统。

成功！

如果直接导入已有的ext4.vhdx，可能会失败。

git

  1. 官网下载，这里下载的是64-bit Git for Windows Portable便携版。

  2. 双击安装到指定目录D:\develop\git\PortableGit\

  3. 创建全局配置文件D:\develop\git\.gitconfig

  4. 创建.ssh文件目录D:\develop\git\.ssh

  5. 设置环境变量

    1. 设置环境变量GIT_HOME，值D:\develop\git
    2. 设置环境变量GIT_CONFIG_GLOBAL，值%GIT_HOME%\.gitconfig
    3. 在Path中添加，%GIT_HOME%\PortableGit\bin

  6. 创建ssh密钥

    1. 生成公钥私钥

      记得在生成过程中，指定生成目录D:\develop\git\.ssh，否则可以去根目录User\yourname\.shh中复制到指定目录

    2. 指定密钥文件

  7. 设置全局用户

  8. 设置获取.gitignore文件别名

    使用示例

    将内容复制到.gitignore文件即可

  9. 测试

    • git -v

go

go环境

  1. 下载包，官网下载地址，这里是下载压缩包go1.22.2.windows-amd64.zip。

  2. 解压并复制到指定目录。

  3. 配置环境变量。

    • 设置环境变量GO_HOME : D:\develop\go
    • 在Path中加入 %GO_HOME%\go_path\bin和%GO_HOME%\package\go1.22.2\bin。

  4. 设置go环境。
